# Sensitive Dictionary Sources

Source: `konsheng/Sensitive-lexicon`

- Repository: https://github.com/konsheng/Sensitive-lexicon
- License: MIT
- Retrieved: 2026-07-11
- Included scope: every `.txt` file from the repository's `Vocabulary` directory.
- No vocabulary from another repository or website is included.

The generated dictionary excludes one-character entries, pure ASCII entries shorter than four characters, duplicates, and trailing `*` markers.
